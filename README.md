## Description:
RSS Aggregator is a web application for automatically collecting messages, news, articles, and updates from various websites, blogs, and channels in one place. It allows users to read content from multiple sources in a single interface without having to visit each site separately. 
The RSS aggregator allows users to add and manage multiple RSS feeds, automatically update feeds in real time, and read content using a simple and user-friendly interface.

## Features
* ✅ URL validation for RSS feeds
* ✅ Real-time feed updates
* ✅ Bootstrap-powered responsive UI
* ✅ Multilingual support
* ✅ Detailed error handling
